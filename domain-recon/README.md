#### What This Is
This part focuses on collecting basic information about a target domain.

#### What I Did
### Using crt.sh
I used crt.sh to find subdomains from SSL certificates.
this is useful because sometimes subdomains are exposed through certificates even if they are not publicly listed.

##### Using BuiltWith
this helped me understand what technologies a website is running (like frameworks, CMS ...)


##### Using VirusTotal
